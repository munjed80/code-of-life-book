# code-of-life-book

مستودع نصي لإدارة كتاب طويل حول: **الكود، السنن، السببية، فعل الإنسان، اختيار النتيجة، الصحيفة، التوبة، القضاء والقدر، والعمل الصالح**.

## الهدف من هذا المستودع
- تنظيم كتابة الكتاب على مراحل واضحة.
- فصل المحتوى الأساسي عن المسودات والمراجع.
- تسهيل المراجعة والتحديث والتتبع عبر GitHub.

## هيكل المشروع
- `BOOK_RULES.md` : قواعد ومنهجية كتابة الكتاب.
- `GLOSSARY.md` : معجم المصطلحات الأساسية.
- `OUTLINE.md` : المخطط العام للكتاب والفصول.
- `QURAN_EVIDENCE.md` : أدلة قرآنية مرتبة بحسب الموضوع.
- `SCIENCE_NOTES.md` : ملاحظات علمية داعمة للأفكار.
- `IBN_TAYMIYYAH_NOTES.md` : خلاصات واقتباسات مرتبطة بموضوع الكتاب.
- `chapters/` : الفصول الأساسية (12 فصلًا).
- `drafts/` : مسودات العمل والأفكار غير النهائية.
- `references/` : المراجع والمصادر.

## طريقة العمل المقترحة
1. ابدأ بتحديث `OUTLINE.md` قبل كتابة أي فصل.
2. اكتب كل فصل في ملفه داخل `chapters/`.
3. انقل الأفكار الأولية إلى `drafts/` ثم صقّلها قبل نقلها للفصول.
4. اربط كل فكرة رئيسية بدليلها من `QURAN_EVIDENCE.md` أو المراجع.
